## 📖 Descripción del Proyecto

Este repositorio contiene la propuesta de diseño para la Situación de Aprendizaje (SdA) **"Misión Nube Segura"**.

En este proyecto, el alumnado asume el rol de un equipo de administradores de sistemas que debe desplegar una infraestructura segura para una empresa utilizando **Amazon Web Services (AWS)** (creación de una VPC y configuración de instancias EC2).

El núcleo metodológico del proyecto se basa en la introducción de un **"fallo controlado"**: la preparación de un entorno de red restrictivo donde el puerto 22 (SSH) está bloqueado por un firewall. El alumnado deberá realizar labores de **troubleshooting** y aprender a implementar un **servidor VPN** para lograr una conexión remota segura mediante tunelización.

Finalmente, el proyecto culmina con un debate de **reflexión crítica** sobre los dilemas éticos y las consecuencias de utilizar VPNs para evadir políticas de seguridad corporativas o educativas.

## 🧠 Justificación Pedagógica (Indicadores MRCDD)

El diseño de esta SdA emana directamente del análisis de la práctica docente mediante el Ciclo Reflexivo de Gibbs (Tarea 1.1), abordando los siguientes indicadores del **Marco de Referencia de la Competencia Digital Docente (Nivel B2)**:

- **Gestión de la Incertidumbre:**  
  Se fomenta el desarrollo de la resiliencia tecnológica y la autonomía del alumnado, no evitando los errores, sino provocándolos de manera didáctica mediante restricciones de red reales.

- **Dilemas Éticos:**  
  Se integra un espacio obligatorio de reflexión y debate sobre el uso legítimo de herramientas de evasión de red y el respeto a las políticas de ciberseguridad corporativas.

- **Creación de Secuencias de Aprendizaje:**  
  Diseño estructurado y coherente adaptado al contexto del CFGS ASIR.
